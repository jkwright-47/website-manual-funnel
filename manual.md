## 4. Setup

**Step 1.** Create a folder on your desktop named `my-website`.

**Step 2.** Open a text editor. Use one of these (free):
- [VS Code](https://code.visualstudio.com) — recommended
- Notepad (Windows) or TextEdit (Mac)

**Step 3.** Inside `my-website`, create three files:
- `index.html`
- `styles.css`
- `script.js`

Leave them empty for now.

---

## 5. Build the Website

**Step 1.** Open Claude at [claude.ai](https://claude.ai).

**Step 2.** Use this prompt:

> "Build me a clean, professional landing page for [describe your business or client's business]. Include a hero section with a headline and call-to-action button, a short about section, a services section with 3 items, and a contact section with a form. Use HTML, CSS, and JavaScript. Make it mobile responsive."

**Step 3.** Claude will return the code. Copy it.

**Step 4.** Paste the HTML into `index.html`. If Claude splits CSS and JS into separate blocks, paste those into `styles.css` and `script.js`.

**Step 5.** Save all files.

---

## 6. Preview the Site

**Step 1.** Open `index.html` in your browser by double-clicking it.

**Step 2.** You'll see your website.

**Step 3.** Resize the browser window to check it looks good on mobile.

If something looks off, keep going — you'll fix it next.

---

## 7. Improve the Design

Go back to Claude and use targeted prompts to fix specific things:

- "Make the font larger and easier to read."
- "Change the color scheme to dark navy and white."
- "Make the navigation sticky so it stays at the top when scrolling."
- "Add a smooth scroll animation when clicking nav links."
- "Make the contact form validate the email before submitting."

Copy the updated code. Replace what's in your file. Save. Refresh the browser.

Repeat until it looks right.

## 8. Install Git

Git lets you publish your site to GitHub Pages for free.

**Mac:**

1. Open Terminal (search for it in Spotlight).
2. Type: `git --version`
3. If Git isn't installed, follow the prompt to install it.

**Windows:**

1. Go to [git-scm.com](https://git-scm.com) and download the installer.
2. Run the installer with default settings.
3. Open Git Bash after installation.

**Verify it works:**

```
git --version
```

You should see a version number.

---

## 9. GitHub Setup

**Step 1.** Create a free account at [github.com](https://github.com).

**Step 2.** Click **New Repository**.

**Step 3.** Name it: `my-website`

**Step 4.** Set it to **Public**.

**Step 5.** Click **Create Repository**.

**Step 6.** Follow the instructions GitHub shows to push an existing folder.

---

## 10. Make the Site Live

**Step 1.** In your GitHub repository, go to **Settings**.

**Step 2.** Click **Pages** in the left sidebar.

**Step 3.** Under **Source**, select **Deploy from a branch**.

**Step 4.** Choose `main` branch and `/ (root)`.

**Step 5.** Click **Save**.

**Step 6.** Wait 1–2 minutes. Refresh the page.

Your site will be live at:

```
https://your-username.github.io/my-website
```

Share that link with anyone.
